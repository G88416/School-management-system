The BIS (Bophelong Independent School) School Management System can now be installed as a mobile app on all types of phones and tablets!

### For Android Devices (Chrome, Edge, Samsung Internet)

1. **Open the website** in your mobile browser (Chrome, Edge, or Samsung Internet)
2. **Look for the install prompt** at the bottom of the screen that appears automatically
3. **Tap "Install"** button on the prompt
4. **Alternatively:**
   - Tap the **menu icon** (three dots) in the browser
   - Select **"Add to Home Screen"** or **"Install App"**
   - Confirm the installation

### For iOS Devices (iPhone & iPad)

1. **Open the website** in Safari browser
2. **Tap the Share button** (square with arrow pointing up) at the bottom
3. **Scroll down** and tap **"Add to Home Screen"**
4. **Edit the name** if desired (default: "BIS Management")
5. **Tap "Add"** in the top right corner
6. The app icon will appear on your home screen

### For Desktop (Windows, Mac, Linux)

#### Chrome / Edge:
1. **Open the website** in Chrome or Edge browser
2. **Look for the install icon** (âŠ• or computer icon) in the address bar
3. **Click the icon** and select **"Install"**
4. **Alternatively:**
   - Click the **menu icon** (three dots)
   - Select **"Install [App Name]"** or **"Add to Desktop"**

#### Safari (Mac):
1. **Open the website** in Safari
2. **Click Share** button in the toolbar
3. Select **"Add to Dock"**

## ğŸ› ï¸ Troubleshooting

### App Install Prompt Not Showing?
1. Make sure you're using a supported browser
2. The site must be served over HTTPS (not HTTP)
3. Try clearing browser cache and reloading
4. On iOS, you must use Safari browser

### Can't Install on iOS?
- Only Safari browser supports "Add to Home Screen" on iOS
- Make sure you're using the Share button (not the menu)
- iOS doesn't show automatic install prompts like Android

### App Not Working Offline?
- First visit requires internet to cache resources
- Some features require internet connection
- Service worker needs time to cache on first load
